## 3. Detail on how to run the application

### 1. Make sure you have completed the React Native - Environment Setup instructions.
### 2. Start the Metro Server.
```bash
# using npm
npm start

# OR using Yarn
yarn start
```
### 3. Start Application.
### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
